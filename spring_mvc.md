# Spring MVC

This file gives all the information about Spring MVC

## Prerequisites

- Maven
- Apache Tomcat
- Apache CXF (for SOAP web services)

## Setup

- ```New``` > ```Maven Project```
- ```Archetype``` > ```maven-archetype-webapp```
  - ```Archetype ID``` -> name of the project
  - ```Group ID``` -> anything
  - ```Package``` -> name of the package
